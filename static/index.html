Mandrill Template Translation

This is a SparkPost to Mandrill template translation service. From here you can translate a template from file or migrate one directly from your Mandrill account into SparkPost.

BETA: This is pre-release software. Join us on Slack if you have questions or run into issues.

Supported Features

  • Handlebars syntax
  • Logic and iteration: if, each, ...

Unsupported Features

  • Mailchimp syntax
  • HTML escaping: put your markup in your template ;)
  • Inline helpers: upper, lower, title, url, date, striptags
  • Block helpers: unless, with
